Advanced Tutorials
==================

This section is for users who want to go beyond end-to-end model examples and
start customizing ``nkDSL`` itself.

You will learn how to:

* create and register custom iterator clauses
* create and register custom predicate clauses
* read and debug symbolic IR output
* write piecewise branch logic with conditional emissions
* extend the emission system with custom emission clauses
